# 2026-02-22 — Feature: GitHub Actions CI Workflow

Added a GitHub Actions CI workflow so every push and pull request automatically runs lint, format check, and tests. Previously these gates had to be run manually.

## Details

- **Trigger**: all pushes + PRs targeting `main`.
- **Matrix**: Node 18 and 22 on `ubuntu-latest`.
- **Steps**: `npm ci` (root + server), `npm run lint`, `npm run format:check`, `npm test`.
