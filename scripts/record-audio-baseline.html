M4 P0-5 — Audio Baseline Recorder

Capture the 30s reference clip per megacity. Five cities total. Output as Chrome MediaRecorder WebM/Opus.

How to use

  1. In a separate tab, start the dev server (npm run dev) and open http://localhost:3000.
  2. Toggle audio on (▶). Wait until all seven buses report Loaded.
  3. Pick a city below — the page will fly the map there via the active tab. Do NOT interact during the 10s "static" phase.
  4. At the "drag" prompt, manually drag the map for ~10s.
  5. At the "dwell" prompt, settle on the city center until the announcer triggers.
  6. The 30s recording downloads automatically. Move the WebM into ~/Documents/M4-baseline/. Repeat for the next city.

Cities

Each click captures a 30s clip (10s static + 10s drag + 10s announcer dwell).
